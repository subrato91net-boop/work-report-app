♻️ How to Restore This Backup

If you ever need to restore your data (e.g. moving to Hetzner or a new server), run this command on your PC:

# Step 1 — Install PostgreSQL on your PC (if not installed)
# Step 2 — Run this command:

psql "your_new_database_url" < backup_file.sql

💡 Replace your_new_database_url with the connection string of your new database.
